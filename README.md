2. Datos que recopila la aplicación
Esta aplicación no recopila, no almacena y no comparte datos personales de los usuarios. La aplicación funciona localmente en el dispositivo del usuario.

3. Uso de la información
Los datos introducidos por el usuario en la aplicación se utilizan únicamente para realizar los cálculos solicitados dentro de la propia app y no se envían a servidores externos.

4. Compartición con terceros
Esta aplicación no comparte datos con terceros.

5. Seguridad
Al no recopilar ni transmitir datos personales, no se realiza tratamiento externo de datos del usuario.

6. Conservación y eliminación de datos
La aplicación no conserva datos personales en servidores externos. Cualquier dato introducido se mantiene únicamente en el dispositivo del usuario, si la propia app lo guarda localmente.

7. Menores de edad
Esta aplicación no está diseñada específicamente para menores de 13 años.

8. Cambios en esta política
Esta política puede actualizarse en el futuro. Cualquier cambio se publicará en esta misma página.
